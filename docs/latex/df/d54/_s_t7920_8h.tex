\doxysubsection{Descrição detalhada}
Biblioteca para utilização do display ST7920 com bitbanging. 

Esta biblioteca é uma tradução do código da Controllers\+Tech para Zephyr. (\href{https://controllerstech.com/glcd-128x64-st7920-interfacing-with-stm32/}{\texttt{ https\+://controllerstech.\+com/glcd-\/128x64-\/st7920-\/interfacing-\/with-\/stm32/}})

Essa biblioteca faz um bitbanging de SPI para enviar os dados para o display.
